🌍原理

基本原理

希尔排序在直接插入排序上提出了增量 d 的概念，希尔排序过程如下：

  1. 把序列分为 d 个组，下标相距为 d 的倍数的元素放到同一个组中（d一般初始值为序列长度的一半）。

  2. 对所有的分组进行直接插入排序。

  3. d = d / 2，执行第 1 步，直到 d = 0。
