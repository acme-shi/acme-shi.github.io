🍎原理

将待排序序列划分为 已排序序列 和 未排序序列，对 待排序序列 进行 N - 1 次遍历排序，每次遍历排序的过程如下：

  1. 从 未排序序列 中选择一个最小值 min。
  2. 将最小值插 min 入到 已排序序列 队尾，已排序序列长度加1， 未排序序列长度减1。

时间复杂度: O(n^2)
