冒泡排序

2019-02-11
  • 2019

💣 原理

  • 冒泡排序基本思想：

    1. 将待排序队列分为未排序子队列（位于左侧，初始值为待排序队列）和已排序子队列（位于右侧，初始值为空）。
    2. 走访未排序子队列，选择未排序子队列中的最大值交换到未排序子队列的列尾。
    3. 未排序子队列长度减一， 已排序子队列长度加一。
    4. 未排序子队列长度 == 0 ? 排序结束 ： 重复第2步。

  • 时间复杂度：O(n^2)。

  • 稳定。
